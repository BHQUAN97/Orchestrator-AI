## Findings

### Finding #1 — T01 verify pattern không khớp thực tế

- Task prompt yêu cầu "just a number"
- Verify regex: `\b([0-9]+)\s*async` → yêu cầu format "N async"
- Tất cả 4 model output "0" hoặc "N" trần → FAIL
- **Fix**: nới regex thành `\b([0-9]+)\b` HOẶC sửa prompt yêu cầu format "N async functions"

### Finding #2 — `--no-confirm` không skip budget prompt (BUG)

Log T01 smart:
```
💰 Estimated cost: $0.1064 (range $0.0762 - $0.1217), ~5 iterations
? Proceed? (Y/n)    ← HANG, stdin='ignore'
```
- Runner pass `--no-confirm` + `--budget 0.05` nhưng vẫn hỏi Y/n khi est cost > budget
- Child process hang tới khi timeout → smart runs ngắn bất thường
- **Root cause**: budget warning prompt không respect `--no-confirm` / `ORCAI_BENCHMARK=1`
- **Action**: fix `lib/agent-loop.js` hoặc `bin/orcai.js` auto-skip prompt khi NO_CONFIRM hoặc benchmark env

### Finding #3 — Fixture incomplete cho T01

Log T01 cheap:
```
✓ search "async function" → 0 matches
✓ search "async " → 0 matches
```
- Fixture chỉ copy `lib/agent-loop.js` nhưng search ra 0 async
- Thực tế file có nhiều async
- **Root cause**: search_files/grep có thể chỉ scan workdir root, không recurse `lib/`
- **Action**: verify tool behavior hoặc thêm file khác vào fixture

### Finding #4 — Gemini free quota = 20 req/day

- T05 gemini fail vì "generativelanguage.googleapis.com/generate_content_free_tier_requests, limit: 20"
- Mỗi task cần 3-8 LLM call → 5 task ≈ 20-40 req → HIT LIMIT
- **Action**: dùng Gemini paid tier HOẶC giới hạn model `fast` qua OpenRouter fallback

### Finding #5 — parseMetrics không parse format thực tế

- Log output: `Tokens: 29444 in, 122 out | cost: $0.0081`
- Regex trong runner.js: `/\$([\d.]+)\s*(total|cost|usd)/i` → match nhưng không capture đúng field `cost_usd`
- **Action**: sửa regex thành `/cost:\s*\$([\d.]+)/i`

### Finding #6 — Router `fast` model không ổn định

- Call 1: routed Gemini direct → OK
- Call 2: routed OpenRouter → FAIL (credit insufficient lúc đó)
- `usage-based-routing-v2` không pin deployment
- **Action**: đặt Gemini direct làm primary với priority cao, OpenRouter fallback khi 429
